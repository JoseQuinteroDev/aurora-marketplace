# Aurora Marketplace Frontend

Angular + Tailwind storefront/admin shell for Aurora Marketplace.

## Commands

```powershell
npm.cmd install
npm.cmd start
npm.cmd run build
```

The development server proxies `/api` and `/actuator` to `http://localhost:8080`.

## Routes

- `/`
- `/catalog`
- `/products/:slug`
- `/login`
- `/register`
- `/admin`
